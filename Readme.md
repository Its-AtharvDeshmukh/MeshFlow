# 🚀 MeshFlow X

> **The Intelligent Multi-Model AI Operating System**

MeshFlow X is an AI Operating System built for the **Mesh API Hackathon**.

Instead of sending an entire document to a single Large Language Model, MeshFlow X intelligently understands the document, decomposes it into multiple specialized tasks, routes each task to the most suitable AI model through **Mesh API**, executes a Directed Acyclic Graph (DAG), and synthesizes all outputs into a single executive-quality report.

# 🧠 Intelligent Routing

MeshFlow X does **not** use a single AI model.

Each workflow node is evaluated independently.

The router compares multiple candidate models based on:

- Capability
- Speed
- Cost
- Context Window
- Task Type
- Availability

The best model is selected automatically through **Mesh API**.

Example providers include:

- OpenAI
- Google
- Anthropic
- DeepSeek
- Qwen
- Meta
- Mistral
- Cohere
- Grok
- GLM

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Its-AtharvDeshmukh/MeshFlow.git
```

Go into the project

```bash
cd MeshFlow
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
MESH_API_KEY=your_mesh_api_key
PORT=3000
```

Run the project

```bash
npm start
```

or

```bash
npm run dev
```

Open

```
http://localhost:3000
```
